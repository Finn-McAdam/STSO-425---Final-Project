Canada circa 2023

I feel that gathering and calculating the data went very well. I later found I could have gotten older data but 1970 seems a good cutoff point. I also think my averaging was accurate.

I decided to do a website as I like making stuff like websites and its a simple way to show stuff. I used graphs because they are quick and easy to do with a website, which can often take a long time to make.

The biggest problems came from the python code. I chose to use python as my data was in .csv files which are easy to work with in python. However the data I got was split between each station so to get the data I wanted I had to average between all the stations, then average for the whole year as I had yearly fire data but monthly weather data. The math was surprisingly difficult due to my code being a bit of a mess.
If I did this again some better splitting of each process in the code, so that the total average and the average for each territory were done in sepereate function would help reduce the mess. As well as possibly splitting the average for each month and the average for each year into seperate functions.

One thing I wish could have been done is graphs of data for each individual provinces and territory. However, that would have been time consuming and I decided to spend my time elsewhere. A map to show where the fires and temperatures were would also be nice but that would require splitting the data into individual provinces first

I think this definitely connects with how simplifing data can make it hard to see what is really going on, shown in class in Seeing Like a State by James Scott. This is best shown with the drop in temperature around the time there are large and damaging fires. The only thing my data could show was that that result is not connected to data being missing (see the null values graph on the right). My best guess is what my parents suggested, that smoke and ash from the fires led to a reduction in temperatures. However, because I focused on just the temperature and rainfall data, I can't support or deny that. I think I would need data on ash and smoke from the fires and whether those would affect temperature to find out.
