Sources for my data

  • I got my data on wildfires from the Canadian government's fire history data explorer
  • I got my weather data from the Canadian government's monthly climate summaries

Both of these are using data between 1970 and 2024

Tools used to interpret the data

  • I used Google Sheets to create the graphs
  • I made some python code to take out the climate data that was useful to me and put it in a format I can use. The code is currently up on Github
